       ```tsx

             Unload Model
           </Button>
         </div>
       )}
       ```

       All sliders wrapped in a div with `className={cn("space-y-4", !isModelLoaded && "opacity-50 pointer-events-none")}`:

       - **Pitch Shift** (f0UpKey): Range -12 to +12, step 1, default 0. Label shows current value with "semitones" suffix.
         ```tsx
         <div>
           <Label>Pitch Shift: {settings.rvc.f0UpKey} semitones</Label>
           <input type="range" min={-12} max={12} step={1}
             value={settings.rvc.f0UpKey}
             onChange={(e) => {
               const val = parseInt(e.target.value)
               settings.updateRVC({ f0UpKey: val })
               sendMessage('rvc_set_params', { f0_up_key: val })
             }}
             className="w-full" />
         </div>
         ```

       - **Filter Radius** (filterRadius): Range 0 to 7, step 1, default 3. Pitch smoothing.

       - **Resample Rate** (resampleSr): Select dropdown with options: 0 (disabled), 16000, 22050, 24000, 32000, 40000, 44100, 48000. Default 0.

       - **Volume Envelope** (volumeEnvelope): Range 0.0 to 1.0, step 0.05, default 0.0. Volume envelope mixing.

       - **Protect Consonants** (protect): Range 0.0 to 0.5, step 0.01, default 0.33. Higher = more consonant protection.

       - **RMS Mix Rate** (rmsMixRate): Range 0.0 to 1.0, step 0.05, default 0.25. Loudness matching.
